AI Chatbot using HTML, CSS, JavaScript & Python Flask

This project is a modern AI chatbot web application built using HTML, CSS, JavaScript, Python Flask, and the Google Gemini API.

The chatbot allows users to interact with an AI assistant through a beautiful and responsive web interface.
The project was developed step-by-step by learning:

Basic chatbot input/output flow
AI response generation using Gemini API
Chat memory handling
Frontend chatbot UI design
Backend integration using Flask
API communication between frontend and backend
Responsive chatbot layout
